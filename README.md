# FNP-Sales-Analysis-Excel
An interactive Excel dashboard analyzing ₹35.2L in sales data, uncovering seasonal trends, and tracking KPIs for FnP


## Project Overview
This project features an interactive Data Analysis Dashboard created entirely in **Microsoft Excel** for FnP. It is designed to track Key Performance Indicators (KPIs), uncover seasonal purchasing trends, and facilitate data-driven decision-making for inventory and logistics. 


## Key Highlights & Insights
* **Revenue & Order Tracking:** Analyzed a dataset of 1,000 total orders, representing a total revenue of ₹35.21 Lakhs, with an average customer spend of ₹3,520.98.
* **Geographic & Operational Metrics:** Tracked order volumes across the top 10 cities to understand regional demand and monitored operational efficiency by calculating the average time between order and delivery (5.53 days).
* **Temporal & Seasonal Trends:** Mapped revenue trends across months and specific hours of the day to identify peak ordering times. Visualized revenue spikes tied to key occasions like Anniversaries, Valentine's Day, and Diwali.
* **Product & Category Performance:** Categorized sales data to highlight top-performing segments, revealing strong revenue drivers across categories like Colors, Soft Toys, and Cakes.

## Tools & Skills Used
* **Microsoft Excel:** PivotTables, PivotCharts, Advanced Formulas, and Slicers.
* **Data Analytics:** Data cleaning, transformation, and exploratory data analysis (EDA).
* **Data Visualization & Business Intelligence.**

## How to Interact with the Dashboard
1. Download the `FnP Sales Analysis Project.xlsx` file from this repository.
2. Open the file in Microsoft Excel.
3. Use the interactive slicers on the right-hand side to filter the dashboard by **Order Date**, **Delivery Date**, or specific **Occasions** (e.g., Diwali, Birthday).
